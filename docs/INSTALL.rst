reddplusid.mcpolicy Installation
--------------------------------

To install reddplusid.mcpolicy using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``reddplusid.mcpolicy`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        reddplusid.mcpolicy

* Re-run buildout, e.g. with:

    $ ./bin/buildout
